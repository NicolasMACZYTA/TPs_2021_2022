Sujet du projet

Le but du projet est de développer une suite logicielle permettant de manipuler les expressions régulières, les automates et les grammaires. Les outils doivent implémenter les méthodes de construction et de transformation, ainsi que les algorithmes vus en cours. Le projet peut être réalisé en binôme. TOUS LES ETUDIANTS doivent répondre au feedback "Groupes projet".

La notation

La note du projet sera basée sur le nombre de fonctionnalités implémentées (et leur bon fonctionnement), sur le respect des consignes et du cours. Il n'est pas obligatoire de les implémenter toutes, en sachant que certaines sont plus simples que d'autres.

Le projet doit contenir un makefile. Les fichiers exécutables doivent être placés à la racine du projet (dans le même répertoire que le makefile). Ils devront porter le nom spécifié dans le sujet. Un fichier _lisezMoi.txt devra être présent, suivant le format imposé : son but est d'indiquer les fonctionnalités développées.

Les fonctionnalités demandées

Les automates et les expressions régulières
Visualisation d'automates

L'outil auto2latex permet de convertir un automate au format .aut en un fichier au format LaTeX qui utilise la bibliothèque tikz. L'objectif étant de visualiser l'automate. Le fichier tex doit pouvoir être compilé pour générer un PDF, par exemple.
Syntaxe : ./auto2latex fichier.aut fichier.tex

Déterminisation d'un automate

L'outil autodet permet de déterminiser un AFN à l'aide de l'algorithme vu en cours. Un fichier LaTeX est généré, contenant les étapes de l'algorithme et l'automate généré.
Syntaxe : ./autodet fichierIn.aut fichierOut.aut visu.tex

Minimisation d'un automate

L'outil automin permet de minimiser un AFD à l'aide de l'algorithme vu en cours. Un fichier LaTeX est généré, contenant les étapes de l'algorithme (le tableau) et l'automate généré.
Syntaxe : ./automin fichierIn.aut fichierOut.aut visu.tex

De l'automate vers l'expression régulière

L'outil auto2exp permet de convertir un automate en une expression régulière à l'aide de la méthode de Brzozowski et McCluskey. Chaque étape est affichée dans un fichier LaTeX.
Syntaxe : ./auto2exp fichierIn.aut fichierOut.exp visu.tex

De l'expression régulière à l'automate

L'outil exp2auto permet de convertir une expression régulière en un AFN à l'aide de la construction de Thomson. Chaque étape est affichée dans un fichier LaTeX à l'aide de la bibliothèque tikz.
Syntaxe : ./exp2auto fichierIn.aut fichierOut.aut visu.tex

Exécution

L'outil execauto permet d'exécuter un mot par un automate (AFD, AFN ou AFNP). La sortie correspond à un fichier LateX.
Syntaxe ./execauto fichierIn.aut fichierIn.txt visu.tex

Les grammaires
Exécution

L'outil typegram permet d'afficher le type de la grammaire.
Syntaxe ./typegram fichierIn.gra

Elimination de la récursivité gauche immédiate

L'outil recurimgram permet de supprimer la récursivité gauche d'un symbole non terminal d'une grammaire.
Syntaxe ./recurimgram fichierIn.gra A fichierOut.gra (avec A un symbole non terminal)

Elimination de la récursivité gauche profonde

L'outil recurgram permet de supprimer la récursivité profonde d'une grammaire.
Syntaxe ./recurimgram fichierIn.gra fichierOut.gra

Factorisation

L'outil factgram permet de factoriser une grammaire en fonction d'un symbole non terminal spécifié.
Syntaxe ./recurimgram fichierIn.gra A fichierOut.gra (avec A un symbole non terminal)

Ensembles Premier et Suivant

L'outil premsuiv permet de calculer les ensembles Premier et Suivant d'une grammaire
Syntaxe ./premsuiv fichierIn.gra fichierOut.tex

Analyseur LL

L'outil LLgram permet de créer un analyseur d'une grammaire LL. Un fichier LaTeX est généré permettant de visualiser l'analyseur.
Syntaxe ./LLgram fichierIn.gra fichierOut.ll fichierOut.tex

Exécution d'un analyseur LL

L'outil execLL permet d'exécuter un analyseur LL sur un mot d'entrée. La sortie est générée dans un fichier LaTeX.
Syntaxe ./execLL fichierIn.ll fichierIn.txt fichierOut.tex

Analyseur SLR

L'outil SLRgram permet de créer un analyseur SLR d'une grammaire. Un fichier LaTeX est généré permettant de visualiser l'analyseur.
Syntaxe ./SLRgram fichierIn.gra fichierOut.slr fichierOut.tex

Exécution d'un analyseur SLR

L'outil execSLR permet d'exécuter un analyseur SLR sur un mot d'entrée. La sortie est générée dans un fichier LaTeX.
Syntaxe ./execSLR fichierIn.slr fichierIn.txt fichierOut.tex

Analyseur LR(1)

L'outil LR1gram permet de créer un analyseur LR(1) d'une grammaire. Un fichier LaTeX est généré permettant de visualiser l'analyseur.
Syntaxe ./LR1gram fichierIn.gra fichierOut.lr1 fichierOut.tex

Exécution d'un analyseur LR(1)

L'outil execLR1 permet d'exécuter un analyseur LR(1) sur un mot d'entrée. La sortie est générée dans un fichier LaTeX.
Syntaxe ./execLR1 fichierIn.lr1 fichierIn.txt fichierOut.tex

Analyseur LALR(1)

L'outil LALRgram permet de créer un analyseur LALR(1) d'une grammaire. Un fichier LaTeX est généré permettant de visualiser l'analyseur.
